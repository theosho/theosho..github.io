  • Web Hosting Control Panel

A crucial component of our shared web hosting packages platform is our in–house developed Control Panel. It’s the only Control Panel you have to sign into in order to manage your Internet presence, which eliminates the dependency on any other interfaces. You can seamlessly manage your web sites, your domains and your payment transactions from a single location.

Our Control Panel is designed to make it easier for you to administer your website. It’s simple to browse through and you can easily navigate from one section to another no matter where you’re. All the tools that we’ve built are intuitive and user–friendly, so there’s no real learning curve. And we have included numerous other charge–free tools that can help you with any project – from speeding up and enhancing your existing websites to launching brand new web sites.

Email Manager

E–mail management is really easy with us

When you own a website, you need to also administer all the email boxes pertaining to it. And while composing emails is really easy, managing the respective email accounts can be tough. You may need to cope with unsolicited bulk electronic mails, to create email filters, to worry about security issues, etc.

With our Email Manager, you will discover that managing email mailboxes is not difficult at all. You’ll have quick access to all critical functions (you can forward electronic mails, set up mail filters, enable unsolicited bulk email protection, etc.) and you can administer multiple email box accounts at once.

File Manager

File management in the digital age

Web file managers have been around for a long time. But they’re clunky, slow, difficult to work with and lacking in features. We have changed the situation. Our File Manager behaves more like a desktop file manager than a standard web file manager. You can upload files by dragging & dropping them, or move files and folders from one place to another. You can resort to our inbuilt code editors and preview photographs. You can zip/unzip files or folders directly online.

And everything is done with only a click.

Domain Manager

Managing multiple domains at the same time is very easy

When you have a number of domains, managing them with standard Control Panels can be problematic. The Control Panel–integrated Domain Manager allows you to administer all your domain names at the same time.

If you own multiple domain names, you can also designate different labels to them. This way, you can easily filter your domains and find only the ones that you want.

Dropbox backups

Remote Dropbox backups of your sites

With the Ktm Webhosting Control Panel, you can quickly set up an auto–backup of your sites and your MySQL and PostgreSQL databases in your Dropbox account. This way, you can rest assured that, no matter what happens with our hosting servers, you’ll always have the latest backup at hand.

Our Dropbox data backup service is fully automated – all you need to do is connect your Dropbox account and indicate how many backups you’d like to keep at all times.

A choice of 40+ different apps to install

A choice of 40+ different apps to install

Manual software app installation can be tricky. That is the reason why, we’ve built an automatic Application Installer, which will do the trick for you. It comprises more than 40 well–known web apps and we have made the installation procedure as easy as possible. All you really have to do is specify the administrator username and pass. Our clever tool will do the rest instead of you.

It also maintains a log of the installed software and the used user names and passwords. That way, even if you forget your password, you can always see what it is in our Control Panel.

ModSecurity

A dependable firewall program for your web applications

If you’d like to secure your application against hack attacks, then ModSecurity is here to help you out. It’s a firewall program built to stop hack attacks, cross–site scripting attacks and URL forgery attacks. It operates automatically in accordance with a set of predefined rules and requires no attention from you.

With our Control Panel, ModSecurity will be activated automatically for all domains. You can deactivate it anytime you wish.

Database Manager

A full array of database administration options

The Database Manager located in the Control Panel is a fully featured, yet easy–to–use database administration tool. We’ve simplified the database creation process – you can set up a new database in several seconds. We’ve streamlined the database backup process – you can create a database backup with only one click. The phpMyAdmin and phpPgAdmin tools are also just a mouse click away.

With our Database Manager, you will be able to administer all your databases (MySQL or PostgreSQL) from one single location.

Web Stats Manager

Real–time information on your web site visitors in your Control Panel

Knowing how users interact with their website is vital for any webmaster. With the Web Stats Manager in the Control Panel, you’ll get up–to–the–minute stats about all the things that your visitors do on your site from the moment it is taken online, with no configuration needed on your end.

At any given time, you can see precisely how many people have dropped in on your web site and the average time they have spent on it. You can also view unique and return visits, hits, most–requested webpages, most frequently used key phrases, exit and entry pages, and more.
