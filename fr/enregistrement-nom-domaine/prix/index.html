Contrôle complet de domaines avec Ktm Webhosting

Une évolutivité facile

Changez rapidement votre paquet d'hébergement cloud

Si vous avez un domaine autonome, mais vous désirez également profiter de nos offres d'hébergement cloud, vous pouvez le faire à n'importe quel moment avec quelques clics uniquement. Vous garderez votre panneau de contrôle d'hébergement web et vous pourrez gérer vos domaines, vos sites Internet et vos transactions de facturation à partir d'un seul et unique lieu. En outre, vous pourrez bénéficier de tous les bonus relatifs à nos forfaits d'hébergement cloud.

Noms de domaine Wildcard

Activation de nom de domaine Wildcard par un clic de souris

Si vous avez envie d'utiliser WordPress MultiSite ou toute autre application multi-sites, ou même si vous souhaitez empêcher les utilisateurs de faire une faute de frappe dans la partie ‘www.’ de votre domaine, vous pouvez bénéficier des options d'activation de domaine wildcard offertes par l'interface de notre gestionnaire de domaines.Avec la fonction Wildcard disponible dans le gestionnaire de noms de domaine, vous pourrez faire que tous les hôtes et les sous-domaines sous un nom de domaine donné, par exemple - blog.domaine.com ou forum.domaine.com, se redirigent vers la page d'accueil de votre site web.

Protection WHOIS

Protégez vos infos personnelles de registrant

À partir de votre panneau de contrôle d'hébergement web vous avez la possibilité de protéger facilement les informations whois associées à votre nom de domaine qui par défaut sont publiés sur Internet de façon à ce que tous aient la possibilité de les voir. Demandez tout simplement une activation de la fonctionnalité ID protection pour votre nom de domaine et vos détails de registrant, y compris votre nom, adresse email et adresse postale seront immédiatement remplacés avec des données fictives. Ce service est disponible avec la plupart des domaines de premier niveau génériques et nationaux proposés sur notre site Internet. Dans le gestionnaire de domaines, choisissez tout simplement le domaine que vous désirez protéger et puis touchez le bouton ID Protect pour activer ce service.

Un service client 24h/24 et 7j/7

Une équipe de spécialistes est toujours là pour vous

Nos professionnels expérimentés seront disponibles 24 h/24 afin de vous aider pour tout ennui que vous pourriez rencontrer en gérant vos noms de domaine. Si le domaine que vous voulez enregistrer est déjà occupé ou si vous vous demandez comment activer l'ID protection, contactez-nous tout simplement par un message email ou par un ticket et nous vous répondrons dans l'heure qui suit.Pour recevoir une réponse immédiate, vous pouvez nous appeler ou utiliser l'option live chat pendant les jours ouvrables.

Redirection de domaine

Redirection de domaine effectuée simplement

Si vous devez faire pointer un domaine vers un autre site ou un sous-domaine vers une autre section de votre site Internet, nous avons la meilleure solution pour vous. À l'aide de notre outil convivial de redirection de nom de domaine, vous pouvez effectuer tout en un clic, sans devoir vous occuper avec une redirection .htaccess.Tout ce que vous devez faire pour rediriger votre nom de domaine est de préciser où il devrait mener. Notre système ingénieux fera le reste et votre domaine commencera immédiatement à pointer vers le nouvel emplacement.

Paramétrez des enregistrements de serveurs de noms de domaine personnalisés

Éditez de différentes types d'enregistrements de serveurs de noms de domaine

Nous vous donnons un contrôle total sur les paramètres DNS de vos noms de domaine. Par la section DNS personnalisés du gestionnaire de noms de domaine vous pourrez éditer de différents types d'enregistrements DNS pour un domaine sélectionné. Il s'agit notamment des enregistrements A, AAAA, MX, NS, SRV, CNAME et TXT. Dans le but de paramétrer un nouvel enregistrement, sélectionnez simplement un domaine et puis entrez la nouvelle valeur dans le champ d'enregistrement correspondant. Pour mettre à jour un enregistrement – cliquez sur le bouton ‘Modifier’, pour retourner à la valeur originale – touchez ‘Retourner’.Par l'outil de géolocalisation GeoIP qui est également disponible dans la section DNS personnalisés, vous avez la possibilité de charger une version linguistique particulière de votre site web pour des visiteurs d'un endroit particulier.

Verrouillage de domaine

Protégez vos domaines du vol

Avec Ktm Webhosting, vous pouvez facilement protéger votre domaine contre des transferts non demandés en le “verrouillant” chez votre compagnie d'enregistrement de nom de domaine actuelle. Cela n'affectera en aucune manière la façon par laquelle le domaine marche, mais arrêtera imédiatement toutes les requêtes de transfert de nom de domaine.Et si vous voulez transférer votre domaine, vous pouvez le déverrouiller en un seul clic de souris.

Un outil de redirection GeoIP

Paramétrez facilement des redirections basées sur la géolocalisation

En utilisant l'outil de redirection GeoIP intégré dans le gestionnaire de noms de domaine, vous pouvez pointer une version linguistique donnée de votre site vers un marché géographique particulier. Il est équipé d'une interface claire par pointer-cliquer où vous n'avez besoin d'entrer aucun code. Remplissez simplement l'endroit de vos visiteurs cibles et l'URL que vous souhaitez être chargée par ce groupe de clients.Par exemple, si vous avez une version allemande de votre site sur de.votre-site.com, vous devriez l'entrer dans la case d'adresse URL et puis sélectionnez Allemagne pour un marché de destination. De cette manière, lorsqu'un client allemand charge votre site, il verra de.votre-site.com au lieu de votre-site.com.

Un gestionnaire de noms de domaine

Une solution « tout compris » d'administration de noms de domaine

Le gestionnaire de domaines est une partie intégrante de notre panneau de contrôle et procure tous les outils dont vous avez besoin pour effectuer la gestion de vos noms de domaine à partir d'un endroit. Il a une interface simple où vous pouvez éditer vos enregsitrements de serveurs de noms, changer vos données WHOIS, verrouiller votre nom de domaine ou créer des sous-domaines avec un clic de souris.Par le gestionnaire de domaines, vous pouvez également acheter des services additionnels comme la protection whois et des certificats SSL.

Gestion de multiples noms de domaine

Gestion de plusieurs domaines correctement réalisée

Travailler avec notre gestionnaire de domaines est vraiment simple, que vous ayez un ou de multiples noms de domaine à gérer. Par son interface conviviale, vous pouvez choisir plusieurs noms de domaine simultanément et les prolonger pour une ou plusieurs années, mettre à jour leurs serveurs DNS ou modifier immédiatement les détails whois qui leur sont assignés. Au cas où vous auriez aussi un paquet d'hébergement Internet avec nous, vous pourrez administrer parallèlement de multiples noms de domaine et des sites web à partir d'un lieu.

Parking de domaine

Parking de domaine effectué facilement

Vous pouvez parquer un nom de domaine enregistré en le faisant pointer vers un de nos deux templates de parking de nom de domaine – Pour vendre et En construction. La procédure de parking de domaine est vraiment facile et entrera en vigueur immédiatement. Vous pouvez parquer autant de noms de domaine que vous le souhaitez. Et lorsque vous voudriez déparquer le nom de domaine – vous pouvez le faire avec un seul clic.Vous pouvez également personnaliser nos 2 modèles de parking de nom de domaine en utilisant un texte et des images personnalisés pour les rendre plus personnels.

Certificats de sécurisation SSL

Obtenez des certificats de sécurisation SSL directement à partir de votre panneau de contrôle

La meilleure manière de sécuriser votre boutique en ligne est d'activer un certificat SSL pour votre nom de domaine. Un SSL ajoutera ‘https://’ à votre nom de domaine et montrera à vos utilisateurs potentiels que votre boutique est assez sécurisée. Par la section Gestionnaire de domaines de votre panneau de contrôle d'hébergement web, vous pouvez rapidement activer un certificat générique ou wildcard pour votre site web. Appuyez tout simplement sur le bouton Commander un Certificat SSL, sélectionnez le type de certificat que vous voulez et après cela entrez les détails demandés. Nous nous engageons à vous répondre immédiatement avec vos informations de certificat.
