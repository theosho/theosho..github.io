  • Panneau De Controle Site Web
Le panneau de contrôle d'hébergement Ktm Webhosting est une interface pratique à partir de laquelle vous avez l'opportunité d'administrer toutes les facettes de votre présence sur Internet. Avec de simples actions de "point-and-click", vous avez l'opportunité d'administrer vos noms de domaine, vos sites web et vos transactions de facturation à partir d'un seul et unique emplacement.Notre panneau de contrôle est construit pour travailler uniquement avec notre propre plateforme d'hébergement cloud. Cela le rend plus rapide et beaucoup plus sécurisé que tout autre panneau de contrôle en ligne. Il inclut tous les outils nécessaires pour gérer votre site web, et encore beaucoup d'autres bonus gratuits qui peuvent vous aider plus tard à optimiser votre site existant, ou à rapidement commencer un tout nouveau.

Gestionnaire de messagerie électronique

Administration simple de boîtes de courrier électronique

Le gestionnaire de messagerie électronique intégré dans le panneau de contrôle d'hébergement Ktm Webhosting est disponible pour vous aider à administrer de multiples boîtes de courrier électronique à partir d'un seul et unique emplacement. Il vous fournit une interface conviviale d'où vous pouvez rapidement rediriger des emails, configurer des filtres d'e-mail, activer une protection anti-spam, etc.Si vous utilisez un client mail, vous pouvez aisément configurer une boîte de courrier électronique en utilisant nos options simples d'auto-configuration. Téléchargez simplement le fichier d'auto-configuration pour votre client de courriel et il va effectuer la configuration de votre boîte de courriel automatiquement. Vous pouvez aussi profiter de notre client webmail qui est entièrement équipé et disponible partout dans le monde.

Gestionnaire de noms de domaine

Administrez tout à partir d'un seul endroit

Lorsque vous avez quelques domaines, leur administration avec des panneaux de contrôle habituels pourrait devenir ennuyeuse. Le gestionnaire de noms de domaine intégré dans le panneau de contrôle Ktm Webhosting vous permet de gérer tous vos noms de domaine à la fois.Si vous avez de multiples noms de domaine, vous pouvez aussi leur assigner de différentes étiquettes. De cette façon vous pouvez facilement filtrer vos noms de domaine et trouver uniquement ceux qui vous sont nécessaires.

Gestionnaire de fichiers de Hepsia

La gestion de fichiers web dans le monde numérique

Avec le gestionnaire de fichiers se trouvant dans le panneau de contrôle d'hébergement Ktm Webhosting vous profiterez d'un des gestionnaires de fichiers les plus forts entre vos mains. Vous pouvez télécharger des fichiers en les déplaçant seulement sur votre navigateur. Vous pouvez déplacer des fichiers et des dossiers dans votre compte en les glissant-déposant. Vous pouvez également décompresser des archives téléchargées en un seul clic de souris.Vous pouvez aussi cliquer avec le bouton droit sur n'importe quel fichier ou dossier pour accéder à toutes ces fonctionnalités.

Certificats de sécurisation SSL

Obtenez des certificats de sécurisation SSL directement à partir de votre panneau de contrôle

La meilleure manière de sécuriser votre boutique en ligne est d'activer un certificat SSL pour votre nom de domaine. Un SSL ajoutera ‘https://’ à votre nom de domaine et montrera à vos utilisateurs potentiels que votre boutique est assez sécurisée. Par la section Gestionnaire de domaines de votre panneau de contrôle d'hébergement web, vous pouvez rapidement activer un certificat générique ou wildcard pour votre site web. Appuyez tout simplement sur le bouton Commander un Certificat SSL, sélectionnez le type de certificat que vous voulez et après cela entrez les détails demandés. Nous nous engageons à vous répondre immédiatement avec vos informations de certificat.

Backups Dropbox

Envoyez des backups à votre compte Dropbox de façon automatique

ModSecurity

Un pare-feu sûr pour vos applications

Dans le panneau de contrôle d'hébergement Ktm Webhosting nous avons inclus le module ModSecurity Apache. Il assure un pare-feu pour vos applications, en les protégeant contre des cyber-attaques, fausses injections XSS et demandes de lien URL. Nous avions déjà fixé les règles pour le pare-feu, donc vous ne devez pas le configurer afin qu'il commence à fonctionner.ModSecurity est activé par défaut pour tous vos sites. Ils seront protégés à partir du moment où ils seront lancés sur Internet. Vous pouvez manuellement désactiver ModSecurity pour n'importe quel de vos sites web, si vous le souhaitez.
