Une performance plus rapide

Profitez de l'amélioration des performances de site Web avec notre serveurs semi-dédiés

Nous avions augmenté la bande passante du réseau jusqu'à 10 Go pour offrir une meilleure connectivité et des vitesses accélérées pour vos sites web. Des serveurs additionnels avaient été ajoutés pour simplifier les procédures de maintenance de serveur et pour minimiser les interruptions et les arrêts de disponibilité frustrants du service.Nos serveurs semi dédiés sont aussi équipés avec des disques durs SSD qui se sont avérés plus rapides et résistants que les disques durs réguliers.

Un panneau de contrôle d'hébergement

Gestion de sites intuitive et conviviale

Notre panneau de contrôle par "point-and-click" simplifie la gestion de site Internet et la met au bout de vos doigts. Vous pouvez télécharger des fichiers par de simples actions de glisser-déposer, gérer tous les paramètres de vos domaines et sites web à partir d'un seul endroit, enregistrer et transférer des noms de domaine multiples, paramétrer et administrer de comptes de courrier électronique différents, maintenir des listes de courriel, paramétrer des bases de données MySQL, commander des SSL dans le but de protéger vos visiteurs, etc. Des statistiques détaillées vous renseigneront sur toutes les visites et actions sur vos sites Internet.

Stabilité et fiabilité de service accrues

Serveurs fondés sur Linux avec une sécurité améliorée

Nos serveurs d’hébergement web sont basés sur une distribution Linux assurant une sécurité accrue qui les rend très fiables et stables. Nous avions consacré beaucoup de temps et d'efforts pour le développement d'un réseau sécurisé qui peut résister à des attaques pirates ou à des attaques par déni de service sans produire des interruptions du service d'hébergement Internet et sans compromettre la performance de votre site.

Modèles gratuits

Plus de cent modèles blog/CMS gratuits desquels sélectionner

ModSecurity

Un pare-feu pour application Web

Nous procurons un moyen instantanné de préserver chaque application web dans votre compte, qu'elle soit installée récemment ou qu'elle ait été importée à partir d'une autre compagnie d'hébergement web. Tous nos serveurs semi dédiés sont équipés avec ModSecurity – un service Web Application Firewall qui est activé par défaut. En outre, nous l'avions paramétré pour qu'il résiste de manière automatique à la plupart des attaques de piratage. Cela signifie que dès que vous hébergez votre site sur nos serveurs, il sera plus sécurisé qu'auparavant.

Accès VPN

Navigation privée sûre

Si, pour quelque raison, vous devez cacher vos traces, nous assurons la solution parfaite pour vous. Avec chacun de nos serveurs semi dédiés, vous pouvez recevoir un accès VPN et avoir votre trafic entrant et sortant redirigé à travers un de nos centres de données VPN. En outre, vous pouvez utiliser les mêmes paramètres VPN sur n'importe quel appareil qui est connecté en ligne de bureau, votre ordinateur portatif, votre téléphone, etc.

Backups d'informations

Le contenu web de votre site Internet peut être restauré à tout moment

Aucun site web n'est préservé contre les hacking-attacks ou contre une suppression involontaire du contenu de votre côté. Avec notre service de backup, nous pouvons restaurer le contenu de votre site à tout moment. À côté des sauvegardes du serveur régulières que nous réalisons, vous pouvez vous-mêmes faire des sauvegardes manuelles du contenu de votre site par un clic de souris à partir de la section Gestionnaire de fichiers de votre panneau de contrôle d'hébergement web. Créez tout simplement des archives du(des) dossier(s) que vous souhaitez sauvegarder manuellement et déplacez-les à un lieu à votre choix dans votre compte.

Web accelerators

Augmentez la vitesse de votre site Internet

Avec Ktm Webhosting, vous ne devez pas découper le code ou réduire le contenu de votre site pour que son chargement soit plus rapide. Grâce aux accélérateurs web (Varnish, Node.js et Memcached) intégrés dans le panneau de contrôle d'hébergement web venant avec chaque paquet, vous pouvez faire votre site web plus rapide que jamais – vous pouvez le rendre 10-500 fois plus rapide!

Accès MySQL distant

Connectez-vous à distance à votre base de données

Habituellement, une base de données ne peut être accédée que par de sites qui sont créés dans le même compte que la base de données. Cependant, avec notre option Remote MySQL, vous pouvez donner à une base de données la permission d'accéder à un autre nom d'hôte. Cela peut être très utile si vous avez de différents magasins sur le Web et vous désirez partager une base de données de clients entre eux.
