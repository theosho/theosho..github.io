Cópias de Reserva de Bases de Dados

O conteúdo de seu site pode se recuperar em qualquer altura

Nenhum site é protegido perante ataques cibernéticos ou eliminação inadvertida de dados em sua extremidade. Com o nosso programa de backup, podemos restabelecer o conteúdo do site a qualquer momento. Aparte dos backups de servidores executados regularmente, você mesmo tem a opção de fazer backups manualmente do conteúdo de seu site com um só clique no menu do Gerenciador de Arquivos de seu Painel De Controle. O que precisa fazer é criar um arquivo da(s) a(s) qual(quais) quer copiar manualmente salvando-o num depósito de sua escolha em seu perfil de conta.

Acesso à Distância MySQL

Conectar à base de dados remotamente

De costume, um banco de dados só pode ser acessado por sites que são criados na na mesma conta do banco de dados. Mas por meio da solução MySQL Remoto, é possível permitir a ligação à base de dados para um outro nome de host. A vantagem de uma funcionalidade como esta resulta muito importante se você é dono de múltiplas lojas virtuais e talvez queira distribuir um banco de dados de clientes entre todas essas instâncias.

Cópias de Reserva com o Dropbox

Cópias de reserva automáticas, sincronizadas com sua conta Dropbox

Dropbox é um serviço de armazenagem na Internet utilizado por milhões de pessoas por um motivo – é muito útil, forte, flexível disponibilizado para todos os sistemas operativos. E com a ajuda do Painel De Controle, pode utilizar sua conta Dropbox com o propósito de salvar cópias de segurança recém-feitos de seus websites e bancos de dados MySQL/PgSQL. Para este fim, você precisa conectar sua conta Dropbox para o seu pacote hospedagem semi-dedicada Linux – o nosso agente vai efetuar os backups de maneira automática em base regular.

Acesso VPN

Navegação securizada, privada
