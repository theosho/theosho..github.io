ModSecurity

Um firewall seguro confiável para as suas aplicações baseadas na web

Por padrão, o total de serviços de hospedagem web na nuvem vêm com ModSecurity. ModSecurity apresenta um pequeno elemento do Apache serve de uma barreira de segurança para aplicações web-baseadas segurando efetivamente seu sítio eletrônico contra ataques de hackers. Não faz falta acondicionar ou estabelecer nada em sua extremidade. Assim, o seu site será securizado a partir do momento de alojá-lo em nossa empresa.

Discos de Estado Sólido

Seu sítio eletrônico vai ser mais dinâmico através das unidades de estado sólido

Todos os computadores de secretária e aparelhos portáteis modernos chegam com drives de estado sólido, porque estes se destacam pelas velocidades leitura/escrita muito mais rápidas tornando o desempenho global mais alto. Daí que, os nossos servidores de hospedagem web na nuvem possuem unidades de estado sólido a fim de favorecer os sites alojados conosco para operarem mais celeremente do que se estivessem hospedados em unidades de disco duro convencionais.

Assim, não é necessário mudar, ajustar ou modificar qualquer coisa em seus sítios web, e fazê-los correr mais depressa.

Segurança e Estabilidade Reforçadas do Serviço

Um programa de hospedagem web na nuvem com dupla segurança

Ao gravar programas, correios eletrônicos e bases de dados em alguns aparelhos ao invés de só um, assegurámos uma infraestrutura de hospedagem web na nuvem com muito maior estabilidade para seus sites e aplicações. Também, foi minimizada a vulnerabilidade dos servidores frente a ataques de intrusos/DDoS que tencionam derrubar o servidor nos sistemas comuns de alojamento partilhado.

Aceleradores Web

Faça o seu website incrivelmente mais dinâmico

Os aceleradores web (Varnish, Node.js and Memcached) adicionados ao Painel De Controle foram desenvolvidos para apressar significativamente os seus sítios eletrônicos dinâmicos. Estes são projetados para armazenar no cache os dados de seus websites e, assim, diminuir o número de solicitações ao servidor de bancos de dados ou o servidor API. O que significa um carregamento muito mais rápido dos seus sites permitindo a você chamar a atenção de mais utilizadores felizes, o que equivale a taxas de rejeição muito menores.

Acesso VPN

Navegação protegida, confidencial

No caso de você querer continuar justamente a salvo enquanto navega na web, oferecemos a você uma boa solução. Você pode receber acesso VPN junto com qualquer nos nossos pacotes de hospedagem web na nuvem. Vamos passar todo o tráfego entrante e saliente através de um dos nossos centros de dados VPN, fazendo o seu footprint online indetectável. Poderá ter acesso VPN usando qualquer equipamento ligado à web que você pretende.

Acesso à Distância MySQL

Ligue-se à distância à sua base de dados

Se você tem um banco de dados que precisa usar conjuntamente entre vários websites localizados em muitos servidores e com diferentes fornecedores de alojamento, tire vantagem da nossa opção MySQL à distância. Com este serviço, você já pode habilitar conexões à sua base de dados a quaisquer sites confiáveis. Este é um método cómodo se você é dono de uma base de dados de fregueses que é preciso partilhar entre muitas lojas online.

Gerenciador de Bases de Dados

Um serviço integral de administração de bancos de dados

Com a ajuda do Gerenciador de Bancos de Dados integralizado no Painel De Controle, você será capaz de superintender cada um dos bancos de dados com prontidão e eficácia. Você será capaz construir bancos de dados novinhos com só uns cliques. Através das relações pormenorizadas da utilização dos recursos do MySQL, já saberá você quais são aqueles bancos de dados que têm maior taxa de utilização.

Elaborámos uma aplicação que ajuda você fazer cópias de segurança dos seus bancos de dados com nem mais do que um clique. Deste modo, nem será preciso autenticar-se no menu de de phpMyAdmin.

Vários Centros de Dados

Planos de hospedagem web na nuvem através de 3 continentes

A nossa rede de hospedagem web na nuvem fica distribuída em 3 continentes - América do Norte (o centro de dados Steadfast em Chicago, EUA), Europa (o centro de dados Pulsant em Maidenhead, o Reino Unido; o centro de dados Ficolo em Pori, o Finlândia; o centro de dados Telepoint em Sofia, Bulgária) e Austrália (o centro de dados Amaze em Sydney). Tal distribuição é uma afirmação de que, sem importar onde vive você, ainda será capaz de de se cadastrar n* aquele dos centros de dados que resulte estar mais próximo a os dos clientes em causa. Desta maneira, ficam garantidas as velocidades de carregamento mais rápidas possíveis do sítio eletrônico. Selecione o seu centro de dados favorito mesmo com um clique na a página de registro.
